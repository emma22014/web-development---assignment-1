Web development vs Game development

To be a developer (either Web or game), you need to determine which language you have learn and be professional about it.

If your interest is web developer, PHP and JavaScript can be chosen as main develop language, and advance skills (photoshop, 2D/3D design) will be useful to build a attractive webpage. The first step to become a web developer is learning design layout and adjust details in a website. HTML, CSS and javaScript are basic languages, HTML and CSS need to create surface of webpage, javaScript is the most basic language to develop a webpage. Next step is learning programming language PHP and MySQL database management to create complete webpage. Knowledge in PHP Framework and PHP CMS will be advantage in web developer job, PHP CMS which supports Admin section in a full and professional manner.

About game developer, there are several languages can be use, Java is common language but if you want to develop 3D games, better dont use Java because not many game programmers use so less code sources and examples. C and C++ are programming language is used in nowadays games, these look complicated but logic. All actions will be organized into a list called Function. Objective-C is good language for beginner, it provides simple ways to do complex things. Objective-C++ is superset of Objective-C, when using Objective-C++, you can use C, C++, and Objective-C in the same program.

There are a lot of skills and challenges in web-game development. Based on their interests, developer will choose game world or create website for different businesses. Game and web have it own language, which require developers understand and know how to use and what to do with it.
